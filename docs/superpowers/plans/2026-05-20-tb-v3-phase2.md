# Topology Builder v3 — Phase 2: Scenarios catalog + Lab intent + right-rail scenarios picker

> **For agentic workers:** REQUIRED SUB-SKILL: Use **superpowers:subagent-driven-development** (recommended) or **superpowers:executing-plans** to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship Phase 2 of the v3 Topology Builder revamp on top of the v5.7.0 Phase 1 canvas foundation. Phase 2 introduces the **Lab intent** (chip swaps "Free Build" ↔ "Lab · {title}"), a **right-rail Scenarios picker** (first live rrail icon, slides out to a 320px panel from the right, mirroring Phase 1's Inspector slide pattern), **8 starter scenarios** (one per spec §9 category + 1 extra in Topology types — the full 20-25 catalog is a Phase 2.x follow-up), a **minimal completion validator** ("required devices present + required cables wired" — full L2/L3 reachability ports to Phase 3), and a **Free Build → Lab backup/restore** flow so a user with canvas work doesn't lose it when they load a scenario.

**Architecture:** Additive to the existing `features/topology-builder-v3.js` IIFE (no new feature module — scenarios extracted to a colocated `features/tb-v3-scenarios.js` if cleaner; default plan keeps them inline first, splits if the file crosses ~1500 LOC). One new STORAGE key (`STORAGE.TB_V3_FREEBUILD_BACKUP`). Picker panel slides out from the rrail (same `translateX(100%)→0` mechanism as the Inspector, scoped CSS). Lab intent stored in the same `state` object that ships in Phase 1 (`state.intent` already exists, currently always `'free-build'`).

**Tech Stack:** Vanilla JS (no framework, no build step), SVG for canvas rendering (unchanged), `localStorage` via the existing `STORAGE` wrapper (one new key), CSS appended to `features/topology-builder-v3.css` (loaded via `_ensureCss()` from the feature module, `TB3_CSS_REV` bumped `r4 → r5` for dev cache-bust), Playwright + vm-sandbox for tests.

**Source spec:** [`docs/superpowers/specs/2026-05-20-tb-revamp-design.md`](../specs/2026-05-20-tb-revamp-design.md) — §9 (Scenarios catalog) + §5 (Intent system) + §12 (Phase rollout — Phase 2 row).

**Phase 1 reference plan:** [`docs/superpowers/plans/2026-05-20-tb-revamp.md`](./2026-05-20-tb-revamp.md) — stylistic and pattern reference (the `_fnBody` vm-sandbox UAT extraction, the `_loadFeature` + showPage + open handler sequence, the dogfood-html structure under `.superpowers/brainstorm/17327-1779282193/content/`, the locked Phase 1 lessons).

**Branch discipline:** Feature branch only (`feat/tb-v3-phase2`). No version-bump, no PR, no main push until founder gives ship signal. Each stage commits onto the prior one. The Phase 2 ship triggers v5.8.0.

**Phase 1 lessons folded in:**
1. **`.page` max-width inheritance** — the picker panel is INSIDE the already-full-viewport TB v3 page, so it inherits the Phase 1 fix; the intent-load confirm dialog (if rendered as a modal overlay) needs the same `max-width:none + width:auto + min-width:0` reset accounted for in its scoped CSS block.
2. **`.active` + `:has()` verify** — any verify-eval that drives a feature module's entry path MUST replicate the real handler sequence verbatim (`_loadFeature → showPage → openTopologyBuilderV3`). The `:has(#page-topology-builder-v3.page.active)` selector silently no-ops if `.active` is missing.
3. **CSS rev counter (`TB3_CSS_REV`)** — bump `r4 → r5` in Stage 0 so in-flight dev cache-bust works even while APP_VERSION stays at 5.7.0.
4. **`_deleteSelected` exposure pattern** — when Playwright keypress can't reach a document listener reliably, expose the internal handler on the registration object. Apply this preemptively for `_loadScenario(id)`, `_openPicker()`, `_closePicker()`, `_checkCompletion()`, `_backupFreeBuild()`, `_restoreFreeBuild()`.
5. **Dogfood-html as gate** — Phase 1 had a `phase1-dogfood.html` 8-step founder-run smoke at `.superpowers/brainstorm/17327-1779282193/content/`. Stage 9 plans a parallel `phase2-dogfood.html` for the picker + scenario load + intent swap + completion + backup flow.

```js
  function validateScenarioShape(s) {
    if (!s || typeof s !== 'object') return false;
    if (typeof s.id !== 'string' || !s.id.length) return false;
    if (typeof s.title !== 'string' || !s.title.length) return false;
    var cats = ['topology','architecture','wan','cloud','wireless','security','vlan'];
    if (cats.indexOf(s.category) === -1) return false;
    if (!Array.isArray(s.objectiveRefs)) return false;
    if (!s.startingState || typeof s.startingState !== 'object') return false;
    if (!Array.isArray(s.startingState.devices)) return false;
    if (!Array.isArray(s.startingState.cables)) return false;
    if (!s.completion || typeof s.completion !== 'object') return false;
    if (!Array.isArray(s.completion.requiredDevices)) return false;
    return true;
  }
```

```js
  // Pure: returns the next state after loading a scenario.
  // Caller is responsible for re-rendering canvas + minimap + intent chip + completion pill.
  function loadScenarioOnCanvas(curState, scenario) {
    if (!validateScenarioShape(scenario)) return curState;
    var ss = scenario.startingState;
    return {
      devices: (ss.devices || []).map(function (d) {
        return { id: d.id, type: d.type, x: d.x, y: d.y, label: d.label || '', config: d.config || {} };

      }),
      viewport: ss.viewport ? { x: ss.viewport.x || 0, y: ss.viewport.y || 0, zoom: ss.viewport.zoom || 1 } : { x: 0, y: 0, zoom: 1 },
      intent: 'lab',
      mode: 'design',
      selectedId: null,
      activeScenarioId: scenario.id,
```

```js
  // 'requiredCables' is a pair list — each {from,to} matches if AT LEAST ONE cable
  // connects a device of type 'from' to a device of type 'to' (order-agnostic).
  function checkCompletion(state, completion) {

    var devices = state.devices || [];
    var cables = state.cables || [];

    var typeCount = {};
    devices.forEach(function (d) { typeCount[d.type] = (typeCount[d.type] || 0) + 1; });

    var typeOfId = {};
```

```js
    var pill = document.getElementById('tb3-completion-pill');
    if (!pill) return;

    if (state.intent !== 'lab' || !state.activeScenarioId) {
      pill.hidden = true;
      pill.classList.remove('on');
      pill.textContent = '';
      return;
    }

    var scen = TB_V3_SCENARIOS.find(function (s) { return s.id === state.activeScenarioId; });
    if (!scen || !scen.completion) { pill.hidden = true; return; }

    var res = checkCompletion(state, scen.completion);
    pill.hidden = false;
    if (res.complete) {

      pill.textContent = 'Goals met';
    } else {
```
