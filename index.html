Manual export & import

Exports all quiz history, streaks, drill stats, topology saves, and ACL state to a JSON file. Import merges into your current local data.

Automatic backups

Daily rolling snapshots of every progress key in your browser. Kept for the last 7 days plus any pre-restore safety snapshots. Restore rolls back ALL of your data to that day's state; current state is auto-snapshotted first, so you can roll forward again.
