Design rationale:
  • Eyebrow + headline + sub-line use the same hierarchy as the existing v4.54.0 hero card — consistent with existing visual language.
  • Signal icons: 🎯 = weak spot (recent accuracy gap), 🕒 = stale (untouched). Replaces the “never studied” per-chip stutter — the icon is now the meta.
  • Left-border colour on each chip differentiates the signal type without competing with the dominant purple gradient (yellow accents on dark purple = high contrast, easy to read).
  • Click any chip = drill that topic individually (10-Q quick drill, same affordance as the old chip rows). The primary CTA still wins visually but the secondary path is preserved.
  • Footer composition row shows the math: “2 weak · 3 stale” explicitly answers “which 5 topics?” — the original ambiguity goes away.
  • “Customise →” link (optional, deferrable) lets a power-user drop or swap topics before starting the plan. Dotted underline = subtle secondary affordance.
