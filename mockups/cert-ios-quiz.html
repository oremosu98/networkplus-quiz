Q1 Exam level

A network administrator is deploying a DHCP server and needs to ensure that a critical server always receives the same IP address. Which DHCP feature should be configured?

ADHCP option 66 to specify the TFTP server address
BDHCP reservation to bind the server's MAC address to a specific IP
CDHCP option 51 to extend the lease time indefinitely
DDHCP snooping to prevent unauthorised DHCP servers on the network
Wrong - correct answer: B

DHCP reservation ties a specific MAC address to a fixed IP address within the DHCP scope, ensuring the server always receives the same IP while remaining centrally managed by DHCP. Option A (DHCP option 66) specifies a TFTP server for boot operations, not IP assignment. Option C (DHCP option 51) controls lease time but does not guarantee the same IP, other hosts could receive it. Option D (DHCP snooping) is a security feature that validates DHCP messages, not an assignment mechanism.
