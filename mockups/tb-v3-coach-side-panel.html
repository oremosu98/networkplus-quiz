Design decisions baked in

Position

Right rail, persistent, ~360px wide. Sits alongside Inspector / Picker / Trace / OSI — same panel family.

Mode switching

Canvas state-driven. PBQ active → PBQ body. No tabs, no toggle — the canvas decides the shape.

PBQ entry point

"Start PBQ" CTA inside the existing scenarios catalog. New filter / tab alongside the walkthroughs the PBQs are built off.

AI visibility

Session counter top-right of every panel header. Star (★) marks AI messages; circle (◯) marks scripted. Students always know who's talking.

Routing in UI

PBQ has no free-form input — escape only via the hint cascade. Free Build has the input bar — AI fires on send.

Cost shape

AI fires in exactly two places: FB student questions, PBQ 4th hint. Everywhere else, scripts run. Bounded by design.
