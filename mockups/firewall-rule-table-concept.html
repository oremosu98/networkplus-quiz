Rule · admin access

Ticket 1: the IT admin manages servers over SSH.

Right. The admin needs this path open.

Correct answer: Allow. Ticket 1 grants access.

Right. Scope it to the one admin host, not the whole subnet.

Correct answer: 10.10.10.5/32. Least privilege means just the admin host.

Right. SSH rides TCP 22.

Correct answer: TCP 22. SSH, not RDP or web.

Rule · staff web

Ticket 2: all staff reach the web app over HTTPS.

Right. The whole user subnet is in scope for the web app.

Correct answer: 10.10.10.0/24. Ticket 2 says all staff.

Right. HTTPS is TCP 443.

Correct answer: TCP 443. The ticket names HTTPS.

Rule · catch-all

Ticket 3: everything else from Users to Servers is blocked.

Right. An explicit deny-all backstop closes the VLAN.

Correct answer: Deny · any / any. Ticket 3 blocks everything not matched above.

Arrange the table. The firewall reads from position 1 down and stops at the first match.
1 Deny all any → servers · any
2 Allow admin SSH 10.10.10.5 → .30.20 · 22
3 Allow staff HTTPS 10.10.10.0/24 → .30.10 · 443

Specific allows first, the broad deny last. If the deny sits on top, nothing below it ever runs.
