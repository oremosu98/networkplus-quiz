A technician needs to segment broadcast domains across two switches without adding a router. Which feature accomplishes this?

A Port mirroring
B A VLAN
C Spanning Tree Protocol
D Link aggregation
Why B is correct

A VLAN segments broadcast domains in software, so two switches can carry separate domains over the same hardware — no router required. Routing only matters when those VLANs need to talk.
