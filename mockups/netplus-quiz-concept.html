Which DNS record type is used to map a domain name to an IPv6 address?
Correct

AAAA records map domain names to IPv6 addresses (128-bit). A records map to IPv4 addresses (32-bit). MX records specify mail exchange servers for email routing. CNAME records create aliases for domain names. The question specifically asks for IPv6 mapping, which is the unique purpose of AAAA records.
